# Tantra Espiritual - Website

## Overview
Website profissional de massagens tântricas e terapêuticas. Design sofisticado e feminino com estética budista e tons dourados. Inclui efeitos visuais avançados como mandala rotativa, pétalas de lótus flutuantes, animações de revelação ao scroll e efeitos glassmorphism.

## Structure
- `index.html` - Página principal com todas as seções
- `styles.css` - Estilos CSS completos com animações avançadas
- `app.js` - JavaScript puro para interatividade
- Imagens: about.png, buddha.png, buddha-hands.png, lotus-water.png, espaco1-6.jpg, mandala.png, lotus.png, temple.png, silk.png, favicon.png

## Features
- Seção "Sobre Mim" com biografia profissional
- 11 cartões de serviço com modais detalhados e efeito glassmorphism
- Carrossel de fotos do espaço
- Formulário de contato
- Design responsivo
- Animações suaves e sofisticadas

## Tech Stack
- HTML5, CSS3, JavaScript puro (sem frameworks)
- Fontes: Cormorant Garamond (serif), Raleway (sans-serif)
- Servidor: Python HTTP Server na porta 5000
- Animações CSS: keyframes, transforms, filters
- JavaScript: IntersectionObserver para scroll reveal

## Design

- Text: #f5f5f5 (branco suave)
